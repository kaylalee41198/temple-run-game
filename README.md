# 🏃 Temple Run Game

A Temple Run-style endless runner game built with Next.js, TypeScript, and Canvas 2D rendering.

## Features
- Pseudo-3D road rendering with perspective
- Lane-switching mechanics (3 lanes)
- Jump and slide to avoid obstacles
- Coin collection with score tracking
- High score persistence (localStorage)
- Touch controls for mobile
- Progressive speed increase

## Controls
- **← →** Move between lanes
- **↑ / Space** Jump
- **↓** Slide/Duck
- **Tap** Start/Jump (mobile)
- **Swipe** Move (mobile)

## Tech Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Canvas 2D

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play.

## Deploy
